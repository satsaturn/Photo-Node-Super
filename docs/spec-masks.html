Masks — extension spec

Adds the mask tab to the Edit module's right panel: mask list + per-mask edit list, with multiple shape types. Not in v1; v1 ships the tab as a disabled placeholder (see v1 §3).

Depends on v1 §3 (Edit right-panel tabs), §5 (layer header controls — same per-mask), §6 (pipeline / phase buckets, reused per-mask). A mask's edit list is structurally identical to v1's global edit list; everything in v1 §2, §4, §6, §10 applies per-mask unchanged.

1. Masks

  • A mask = a shape (radial/linear/brush/luminance/etc.) wrapping its own edit list.
  • A mask's edit list is structurally identical to the global one: same built-in sheets available, same 4-phase system, same +/-/dropdown/new-file flow.
  • Masks start empty; user adds only the built-ins/customs wanted for that local adjustment (e.g. local exposure = mask containing the Basic sheet).
  • Principle: a node sheet is identical whether used globally or locally — its phase means the same thing in any edit list, relative to that list's built-ins.
  • Mask-local edits apply only within the masked area.

2. Mask panel location

In the Edit module, the mask panel lives as a tab switcher at the top of the right panel, next to the global tab:

┌─ [global]  [mask]  [crop] ─────────────────────┐
│  ＋ New mask  [radial ▾]                        │
│  ───────────────────────────────────────────── │
│  ☑  Radial 1          · radial   [shape][dup][×]│
│  ☑  Brush sky         · brush    [shape][dup][×]│
│  ☐  Linear grad       · linear   [shape][dup][×]│
│  ───────────────────────────────────────────── │
│  Edit list · Radial 1                            │
│  (same structure as the global edit list)        │
└──────────────────────────────────────────────────┘

3. Mask list

  • New mask — a dropdown/picker at the top of the mask tab lists the available mask shapes (radial, linear, brush, luminance, depth, subject, etc.). Choosing one creates an empty mask with a default name and adds it to the list.
  • Enable toggle — each mask row has a toggle to enable/disable the mask; disabled masks do not contribute to the result.
  • Name — default names are generated from the shape type ("Radial 1", "Brush sky", etc.). The name is editable inline.
  • Type badge — shows the mask shape (radial / linear / brush / luminance / …).
  • Edit shape — opens the mask-drawing controls for that mask on the photo canvas.
  • Duplicate — creates a copy of the mask including its current shape and edit list.
  • Delete — removes the mask after confirmation.
  • Reorder handle — drag to reorder masks in the list; order affects compositing when masks overlap.

4. Selected mask edit list

Selecting a mask in the list loads its edit list beneath the mask list. The edit list has the same structure and rules as the global edit list (v1 §3): built-ins in fixed order, Custom Edits section at the bottom, layer header controls per v1 §5, and the same 4-phase execution rules per v1 §6.
