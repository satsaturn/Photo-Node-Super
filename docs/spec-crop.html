Crop — extension spec

Adds the crop tab to the Edit module's right panel. Not in v1; v1 ships the tab as a disabled placeholder (see v1 §3).

Depends on v1 §3 (Main editor layout) and §6 (pipeline order — crop is bucket 1). In v1 the crop tab is shown disabled with a "coming in a future version" tooltip; the Lens Correction bucket 0 and Crop bucket 1 both execute as identity. This spec enables the crop tab.

1. Crop tab

The crop tab holds the photo's geometry controls. It is a dedicated panel, not a layer in the global edit list.

  • Aspect ratio — presets (Original, 1:1, 4:3, 3:2, 16:9, etc.) plus a free-form option and custom width/height inputs.
  • Straighten — angle slider to rotate the crop, with a level/straighten tool activated from the canvas toolbar.
  • Flip — horizontal and vertical flip toggles.
  • Reset — returns crop, angle, and flip to defaults in one action.

The crop boundary is overlaid on the main canvas. Dragging the canvas overlay adjusts the crop; the right-panel controls update in sync. Crop runs at a fixed pipeline position — see v1 §6 bucket 1.
